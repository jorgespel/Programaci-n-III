## Descripción
Este programa implementa una **lista enlazada simple utilizando memoria dinámica en C++**.  
El usuario puede ingresar números indefinidamente hasta decidir detener el proceso.

Cada número ingresado se almacena en un **nodo creado dinámicamente** y se agrega al **final de la lista**.

Al finalizar el ingreso de datos, el programa recorre la lista y muestra todos los valores almacenados.

Además, el programa calcula:

- Cantidad total de nodos creados
- Suma de los valores ingresados
- Promedio de los valores
