# Sistema de Gestión de Estudiantes 

## Descripción
Este programa permite gestionar estudiantes utilizando una **lista enlazada simple**.

El sistema utiliza memoria dinámica y punteros para almacenar y manipular la información de los estudiantes.

Cada estudiante se guarda en un nodo con los siguientes datos:
- Carné
- Nombre
- Apellido
- Carrera

##Funcionalidades

El programa cuenta con un menú interactivo que permite realizar las siguientes operaciones:

1. **Insertar estudiante al inicio**
   - Agrega un nuevo estudiante al principio de la lista.

2. **Insertar estudiante al final**
   - Agrega un estudiante al final recorriendo la lista.

3. **Mostrar estudiantes**
   - Muestra todos los estudiantes registrados.

4. **Buscar estudiante**
   - Permite buscar un estudiante por su número de carné.

5. **Eliminar estudiante**
   - Elimina un estudiante de la lista utilizando su carné.

6. **Salir**
   - Finaliza la ejecución del programa.
